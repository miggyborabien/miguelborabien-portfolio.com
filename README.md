## 🚀 Getting Started

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/miggyborabien/miguelborabien-portfolio.com.git
   cd miguelborabien-portfolio.com
   ```

2. Open `index.html` in your browser or use a local server:
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Python 2
   python -m SimpleHTTPServer 8000
   ```

3. Navigate to `http://localhost:8000` in your browser.

### GitHub Pages Deployment

This project is configured to deploy automatically to GitHub Pages:

1. Push changes to the `main` branch
2. GitHub automatically builds and deploys to `https://miggyborabien.github.io/miguelborabien-portfolio.com/`

## 🎨 Customization

### Colors
Update the color scheme by modifying CSS variables in `styles.css`:
- Primary: `#2563eb` (Blue)
- Accent: `#60a5fa` (Light Blue)
- Background: `#0f172a` (Dark Blue)

### Content
Edit text and information directly in `index.html`. Key sections to update:
- Hero section title and description
- Services offered
- Experience timeline
- Skills list
- Contact email

### Fonts
Change the font family in `styles.css` or link different Google Fonts in the HTML head.
